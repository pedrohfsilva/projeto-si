# Projeto de Sistemas de Informação — SSC0120 (ICMC/USP)

Repositório de trabalho do grupo. **Tudo o que deve ser entregue no Moodle
está na pasta [`entrega/`](./entrega/README.md).**

## Estrutura do repositório

```
projeto-si/
├── entrega/                          # <-- Arquivos do trabalho escrito
│   ├── README.md                     # Índice da entrega
│   ├── texto_sistematizado.docx      # Texto (50%)
│   ├── texto_sistematizado.pdf       # Versão em PDF do texto
│   ├── infografico.pdf               # Infográfico (10%)
│   ├── infografico.png               # Versão em imagem
│   └── infografico.html              # Fonte do infográfico
│
├── prototipo/                        # Protótipo em Next.js (30%)
│
├── suporte/                          # Scripts e utilitários de suporte
│   ├── gerar_texto_sistematizado.py  # Gera o .docx a partir do conteúdo
│   └── pdf-tool/                     # Puppeteer para gerar o PDF/PNG
│
├── A-Polymerforge.pdf                # Dossiê da empresa-cliente
└── descricao_projeto_sistemas_informacao.md   # Descrição do trabalho
```

Ver [`entrega/README.md`](./entrega/README.md) para detalhes de cada peça,
grupo e declaração de uso de IA.
