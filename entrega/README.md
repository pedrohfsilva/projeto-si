## Executando o protótipo

```bash
cd prototipo
npm install
npm run dev
```

Abrir `http://localhost:3000`.

## Declaração de uso de Inteligência Artificial

Ferramentas de IA foram utilizadas em duas etapas específicas do trabalho:

1. **Protótipo de telas** — o Claude Code (Anthropic) apoiou a codificação do
   protótipo em Next.js, a partir das telas e funcionalidades definidas pelo
   grupo em reunião.
2. **Revisão dos textos** — o Claude Code foi usado para revisar e refinar a
   redação dos textos produzidos pelos integrantes, buscando clareza e coesão,
   sem alterar ideias ou conclusões.

A concepção do problema, a definição da solução, a arquitetura, o mapeamento
de processos e as análises de valor foram elaboradas pelo grupo.
